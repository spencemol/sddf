# SDDF Examples

This directory provides scenario examples for runtime + adapter behavior.

## Runtime-Oriented Examples
- `full-conductor-wave/` - Full-mode wave with mixed backends across parallel workspaces
- `quick-conductor-parallel/` - Multiple independent Quick tasks in parallel workspaces
- `single-workspace-owner-reviewer/` - One mutating owner thread plus advisor/reviewer support
